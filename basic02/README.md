# Typedef 

It is used to give name to some existing type.

 * This name can be used to declare varaibles of that type.
 
 Example
 
```C
typedef int Whole;
Whole amount , numbers;
```
## Structure exmaple

```C
typedef struct{
int date;
int month;
int year;
}; Date;
```
